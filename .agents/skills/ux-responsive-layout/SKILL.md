# 响应式与交互设计规范

执行任何UI开发时，必须遵循以下原子化设计规则：

## 1. 断点与容器适配
- 使用 Tailwind CSS 的 `sm`(640px)、`md`(768px)、`lg`(1024px) 断点。
- **容器宽度**：PC端最大宽度 `1280px` 并居中；移动端 `w-full` 带 `px-4`。
- **高度自适应**：视口高度使用 `min-h-screen`，内部滚动使用 `flex-1 overflow-y-auto`，绝对禁止固定 `height: 100vh` 导致移动端键盘弹出错位。

## 2. 界面焦点（用户看哪里？）
每个页面必须定义 **“主视觉锚点”**：
- **World Explore**：焦点是3D地球（占据70%面积），控件半透明浮动。
- **Journey Focus**：焦点是“进度环 + 下一步任务清单”（占据中间黄金位置）。
- **Unlock Quest**：焦点是十类图标卡片的完成状态。

## 3. 控件显隐逻辑（隐藏什么？）
| 组件类型 | PC端 | 移动端（<768px） |
| :--- | :--- | :--- |
| **来源链接列表** | 全部展开显示 | 折叠为 `📎 N个来源` 气泡，点击弹窗 |
| **地图操作按钮** | 全部平铺（缩放、定位、重置） | 缩为两个核心图标（定位、菜单），其余收入 `+` 号抽屉 |
| **Journey描述** | 显示前300字 | 显示前80字 + “展开” |
| **成员列表** | 头像 + 姓名 + 角色 | 仅显示头像堆叠（Avatar Group） |
| **数据更新时间** | 始终显示完整时间戳 | 显示为 “3分钟前” 缩写 |

## 4. 统一风格（设计Token）
- 主色调：`#F5F0E8`（羊皮纸背景），`#1A3A3A`（深海绿主文字）。
- 圆角：卡片统一 `rounded-xl`，按钮统一 `rounded-full`（胶囊状）。
- 字体：标题使用 `font-serif`，正文使用 `font-sans`。
- 毛玻璃效果：仅用于浮动控制栏，使用 `backdrop-blur-md bg-white/80`。
