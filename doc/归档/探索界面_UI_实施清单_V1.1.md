# Family Atlas 探索界面 UI 实施清单 V1.1

> 施工裁决顺序：用户已确认决策 → 产品设计说明书 → GPT 最终冲突裁决 → Claude UI 工程骨架 → DeepSeek 局部交互参考。  
> `OUR ATLAS` 仅作为界面 Logo；产品正式名称保持 `Family Atlas`。

## Gemini 新增意见裁决

| Gemini 意见 | 裁决 | 实施方式 |
|---|---|---|
| 临近出发 72 小时首页 | 采纳并修正 | 新增 `departure_soon` 模式；默认阈值 72 小时，但必须配置化，不得写死 |
| 目的地进入时静默同步 | 部分采纳 | 数据陈旧时后台自动刷新；手动同步按钮、更新时间和分项进度仍必须保留 |
| 弹性物理磁吸动效 | 部分采纳 | 仅用于卡片、点位和抽屉的轻量微交互；不强制全局固定 `ζ=0.75`；减少动态模式下关闭 |
| AI 语义筛选输入框 | 采纳为 P2 | 通过 Feature Flag 和 Provider Adapter 实现；无 AI 服务时回退到现有手动筛选，不得伪造理解结果 |
| 中文去斜体、对比度、扫描线降噪 | 采纳，已在原规范中 | 维持现有可达性标准，并将 Gemini 记为补充确认来源 |
| 移动端三档 Bottom Sheet | 采纳，已在原规范中 | 维持 25% / 55% / 90% 三档设计 |
| 工具卡固定 85% 透明度 | 不按固定值采纳 | 使用设计 Token，并在约 82%–88% 范围内根据背景和 4.5:1 对比度调整 |

### 外部能力边界

- 目的地天气使用 Open-Meteo；开放、交通、事件、风险在没有已授权 Provider 时明确为“暂不可用”，保留来源、更新时间、手动同步与后续 Adapter 接口。
- 未执行远程 D1 迁移或部署，避免对生产 Cloudflare 资源产生未经授权的写入。
- 自动化验证已覆盖单元/构建；桌面与 375px 的视觉截图待可用浏览器驱动后补录，不能由静态图片或模拟结果替代。
