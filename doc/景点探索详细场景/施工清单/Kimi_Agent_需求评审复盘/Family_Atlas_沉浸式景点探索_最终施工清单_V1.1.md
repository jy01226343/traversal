# FAMILY ATLAS · 沉浸式景点探索 · 最终施工清单 V1.1

用途：项目负责人逐项验收；与《Agent 施工方案 V1.1》配套使用，阶段编号已统一为 **P0–P9**。

> 施工原则：先审计现有仓库，再按阶段施工；复用现有技术栈；不修改已经冻结的产品规则；每个阶段必须通过验收门后才能进入下一阶段。

## V1.1 修订记录（相对 V1.0）

1. 阶段编号与《Agent 施工方案》统一为 P0–P9，并新增"方案章节"交叉引用列。
2. P0 新增性能预算表与埋点方案交付。
3. P1 新增黄金样例三件套、未启用家族与 time_event 降级验收。
4. P2 新增进入中取消、响应式与触控验收。
5. P3 新增空间标签数量上限验收。
6. P7 新增总结页三态标识验收。
7. P8 新增埋点事件验收、无选择默认总结验收。
8. P9 新增性能预算验收、Feature Flag 关闭验收、首期无音频检查。
9. 上线门新增 GATE-09（移动端与埋点）、GATE-10（性能）。

## 使用说明

- 本清单只覆盖首期山地、水域、水下三套场景。
- 每个阶段完成后必须满足验收点，才能进入下一阶段。
- "通过"应以可运行页面、代码文件、测试结果或截图为证据。
- 未通过项目不得用"后续优化"代替；必须明确阻塞原因和处理计划。

## 最终上线门

| 编号 | 验收维度 | 通过标准 |
| --- | --- | --- |
| GATE-01 | 范围 | 仅山地、水域、水下进入首期；无未授权扩展。 |
| GATE-02 | 完整闭环 | 地图进入、五主题探索、总结、业务行动和返回全部可用；进入可取消。 |
| GATE-03 | 可信度 | 实况、典型预览和风险演示全程无混淆，含总结页。 |
| GATE-04 | 风险 | 原因、过程、影响区、信号、行动建议和播放控制完整；仅"注意什么"可触发。 |
| GATE-05 | 水下 | 独立场景、穿水进入、生态、玩法和风险完整。 |
| GATE-06 | 适配 | 无固定星级；结论随人群、日期和玩法改变；无选择时默认规则生效。 |
| GATE-07 | 降级 | 资源失败和减少动态效果时仍能完成旅行判断；FALLBACK 可重试。 |
| GATE-08 | 质量 | 自动化测试、键盘操作、资源清理和回滚验证通过。 |
| GATE-09 | 移动端与埋点 | 移动端视口完成核心闭环；§11.4 事件漏斗可查询。 |
| GATE-10 | 性能 | 预算表各项指标实测达标，或有经确认的豁免记录。 |
